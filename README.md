# Mcontrol - Sistema de gerenciamento de membros
## Descrição
### Contexto

O projeto surgiu atráves de uma necessidade observada no ambiente de uma instituição sem fins lucrativos que necessitava 
cadastrar e gerenciar dados de seus membros, para facilitar a busca por informações e um melhor meio de guardar as informações importantes 
de seus membros.
Inicialmente o projeto visa criar uma aplicação onde será possível cadastrar e editar as informações de pessoas, esse cadastro é realizado
pelo secratário(a) da instituição, que terá o acesso a aplicação no computador da instituição.
Após isso, será realizada uma nova etapa visando adicionar um meio de informar o secretário por meio de um aplicativo de mensagem a ser definido,
sobre aniversário do membro e outras informações

---
### Tecnologias utilizadas:
- Front-end: Vue.js(v3), TypeScript;
- Back-end: Node.js, TypeScript;
- Banco de dados: Prisma ORM, Prisma studio;
